Importar productos en masa

Pega una lista en CSV o JSON para cargar muchos productos de una vez. Es la forma de llenar el catálogo completo sin escribir uno por uno.

Columnas CSV: name, brand, category, price, store, link, image, specs (image y specs opcionales; specs: Pantalla:6.1|RAM:8GB)

Otras formas de cargar todo el catálogo

  • Por archivo: pon tu CSV en server/import.csv y ejecuta npm run import.
  • Automático (desde la web de las tiendas): ejecuta npm run install-browsers y luego npm run crawl; el rastreador lee las categorías de server/catalog-sources.json y agrega los productos solo.
